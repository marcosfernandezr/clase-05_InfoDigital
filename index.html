Rol del SVG en la Infografía Digital

Marcos Fernández

El SVG es uno de los formatos para recursos visuales, cuyas características le permiten ser adaptable y personalizable solo cambiando atributos en su código. Principalmente, los SVG son utilizados para mostrar gráficos no figurativos, los que se centran en mostrar información bajo la convención sobre un tema y la abstracción.

En la presente página se mostrará cómo un gráfico obtenido de RAWGraphs se puede modificar hasta llegar a una interfaz interactiva de manera simple. Además, este formato permite identificar cada elemento del recurso visual para manipularlo a gusto del creador de este. Particularmente, se pueden cambiar tipografías, colores, grosores de fuente, etc.

El gráfico muestra los distintos instrumentos que aparecen en distintos tipos de orquestas. Esto se realiza de tal manera que el elemento principal es la categoría de orquestas y el elemento segundario son los instrumentos. Particularmente, esta relación se muestra con un Sunburst Diagram, lo que permite agrupar de manera simple con el uso de color y pertenencia. Con respecto al color, este permite mostrar de manera más explicita la relación entre orquesta e instrumento. Mientras que el espacio entre instrumentos permite diferenciar cada elemento.

Versión originalmente obtenida de RAWGraphs 2.0 beta. Se utilizan los datos de Orchestras by musical instruments. El gráfico utilizado es Sunburst diagram , el cual se enfoca en mostrar jerarquías y proporciones.
